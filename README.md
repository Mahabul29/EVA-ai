# 🤖 EvaAI Chat

A beautiful, production-ready AI chat web application built with Flask. Features a ChatGPT-like interface and connects to free AI APIs for instant deployment.

## ✨ Features

- 🎨 **Beautiful Dark UI** - Modern ChatGPT-like interface
- 🤖 **AI-Powered Responses** - Connects to free AI APIs (Pollinations AI)
- 💬 **Conversation History** - Persistent chat history with local storage
- 📱 **Responsive Design** - Works on desktop and mobile
- 🚀 **Koyeb Ready** - Pre-configured for Koyeb deployment
- 🔧 **Easy Configuration** - Simple environment variables
- 📝 **Markdown Support** - Code blocks, lists, tables, and more
- 📋 **Copy Code** - One-click copy for code blocks
